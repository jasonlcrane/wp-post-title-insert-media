Adds the post title to the Insert Media screen so you know what post you're editing when adding media.

== Installation ==
You have a couple options:
1. Download the .zip file, unzip it and move the insert-media-post-title folder to your wp-content/plugins folder.
2. Download the .zip file, open your site\'s dashboard, go to Plugins > Add New, then click Upload and select the downloaded .zip file.

Then, go to Plugins > Installed Plugins and activate it.
